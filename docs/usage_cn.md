## 1. 安装与启动

### 1.1 Windows 用户

**最简单的方式 —— 下载 EXE 可执行文件（无需安装 Python）：**

1. 打开浏览器，访问 [Releases 页面](https://github.com/Hotsteel2901/SuperCalculator/releases)
2. 找到最新的版本，点击 `SuperCalculator.exe` 下载
3. 下载完成后，**双击运行** `SuperCalculator.exe`
4. 程序会自动打开一个黑色命令行窗口（用于输出调试信息）和计算器主界面
5. 开始使用！

> 💡 **小贴士：** EXE 文件是独立的，不需要安装 Python 或任何其他依赖，直接双击就能用。

### 1.2 Python 源码运行

**如果你有 Python 环境，可以直接运行源码：**

**第一步：确保 Python 已安装**

打开命令行（Windows 上按 `Win + R`，输入 `cmd`，回车），输入：

```bash
python --version
```

如果显示 `Python 3.9.x` 或更高版本，说明已安装。如果提示找不到命令，请先到 [python.org](https://www.python.org/downloads/) 下载安装 Python 3.9+。

**第二步：安装依赖包**

在命令行中输入：

```bash
pip install numpy matplotlib
```

等待安装完成（通常需要 1-2 分钟）。

**第三步：运行程序**

导航到 SuperCalculator 所在目录，然后运行：

```bash
cd /path/to/SuperCalculator
python super_calc_bridged.py
```

程序会自动检测系统语言，中文系统显示中文界面，英文系统显示英文界面。

### 1.3 Android 用户

**安装步骤：**

1. 打开手机浏览器，访问 [Releases 页面](https://github.com/Hotsteel2901/SuperCalculator/releases)
2. 找到 `SuperCalculator-aarch64.apk`，点击下载
3. 下载完成后，点击 APK 文件进行安装
4. 如果系统提示"未知来源"，请在设置中允许安装（不同手机操作略有不同）
5. 安装完成后，在应用列表中找到 **SuperCalculator** 图标，点击打开

> ⚠️ **注意：** Android 版本需要 Android 7.0 (API 24) 或更高版本。

### 1.4 macOS / Linux 用户

**从源码编译 C 核心库：**

```bash
# macOS
gcc -shared -O2 -fPIC -o calc_core.dylib calc_core.c -lm

# Linux
gcc -shared -O2 -fPIC -o calc_core.so calc_core.c -lm
```

然后安装 Python 依赖并运行：

```bash
pip install numpy matplotlib
python super_calc_bridged.py
```

## 2. 界面概览

启动程序后，你会看到主界面，它由以下几个部分组成：

```
+------------------------------------------+
|  Super Function Graphing Calculator       |
+------------------------------------------+
|  [表达式输入框]   [预设函数下拉菜单]       |
|                                          |
|  [快速输入面板]                           |
|  [常用函数按钮: sin, cos, ln, sqrt...]    |
|                                          |
|  [计算按钮区域]                           |
|  - 求导 / 积分 / 求根 / 极值 / 极限       |
|  - ...更多按钮                           |
|                                          |
|  [结果显示区域]                           |
|                                          |
|  [绘图区域]  (点击"绘图"后出现图表)       |
+------------------------------------------+
```

**界面元素说明：**

- **表达式输入框**：输入数学表达式的地方，例如 `sin(x)` 或 `x^2`
- **预设函数下拉菜单**：内置的常用函数，点击下拉选择即可快速填入
- **快速输入面板**：点击后弹出小键盘，方便快速输入运算符和函数
- **计算按钮**：根据需要选择对应的计算功能
- **结果显示区域**：显示计算结果
- **绘图区域**：展示绘制的函数图形

## 3. 表达式语法

### 3.1 基本运算

| 运算 | 符号 | 示例 | 说明 |
|------|------|------|------|
| 加法 | `+` | `x + 1` | — |
| 减法 | `-` | `x - 3` | — |
| 乘法 | `*` | `2 * x` | ⚠️ 必须写 `*`，不能省略 |
| 除法 | `/` | `x / 2` | — |
| 乘方 | `^` | `x^2` | 表示 x 的平方 |
| 括号 | `()` | `(x+1)^2` | 改变运算优先级 |

**示例表达式：**
- `x^2 + 2*x - 1` → 二次函数
- `sin(x) + cos(x)` → 三角函数
- `sqrt(x^2 + 1)` → 复合函数

### 3.2 函数支持

| 函数 | 说明 | 示例 |
|------|------|------|
| `sin(x)` | 正弦函数 | `sin(x)` |
| `cos(x)` | 余弦函数 | `cos(x)` |
| `tan(x)` | 正切函数 | `tan(x)` |
| `ln(x)` | 自然对数（以 e 为底） | `ln(x)` |
| `log(x)` | 对数 | `log(x)` |
| `exp(x)` | 指数函数 e^x | `exp(-x^2)` |
| `sqrt(x)` | 平方根 | `sqrt(x)` |
| `abs(x)` | 绝对值 | `abs(x)` |
| `floor(x)` | 向下取整 | `floor(x)` |
| `ceil(x)` | 向上取整 | `ceil(x)` |

### 3.3 常数

| 常数 | 值 | 示例 |
|------|-----|------|
| `pi` | π ≈ 3.14159... | `sin(pi * x)` |
| `e` | e ≈ 2.71828... | `exp(x)` = `e^x` |

### 3.4 复数

使用 `i` 表示虚数单位：
- `1+2i` → 复数 1+2i
- `3-4i` → 复数 3-4i
- `abs(3+4i)` → 5（求模）

### 3.5 阶乘与取模

- **阶乘**（后置 `!`）：`5!` = 120，`x!`（x 为非负整数）
- **取模**（求余数）：`10 mod 3` = 1，`7 % 2` = 1

## 4. 函数绘图

### 4.1 基本 2D 函数绘图

**操作步骤：**

1. 在表达式输入框中输入函数，例如 `sin(x)`
2. 点击 **"绘图"** 按钮
3. 程序会在右侧绘图区域显示函数曲线

**示例：绘制正弦函数**

```
输入：sin(x)
点击：绘图
```

你会看到一条优美的正弦曲线出现在绘图区域。

**示例：绘制二次函数**

```
输入：x^2 - 2*x + 1
点击：绘图
```

**示例：绘制指数函数**

```
输入：exp(-x^2)
点击：绘图
```

### 4.2 多曲线叠加

**同时绘制多条曲线：**

1. 在输入框中输入多个表达式，用 **分号** `;` 分隔
2. 例如：`sin(x);cos(x);tan(x)/3`
3. 点击绘图，三条曲线会以不同颜色显示

> 💡 **小贴士：** 可以在输入框输入一条曲线，绘图后再修改为另一条，它们会叠加显示。

### 4.3 坐标标记

**标记曲线上的点：**

1. **左键点击**绘图区域的曲线 → 标记该点坐标
2. **右键点击** → 删除最近的标记点
3. 也可以在输入框中输入具体的 x 值，程序会自动定位并标记对应的点

### 4.4 自定义视图范围

**调整坐标轴范围：**

在界面中可以找到 X/Y 范围设置：
- **X 最小值 / X 最大值**：调整水平范围
- **Y 最小值 / Y 最大值**：调整垂直范围
- **步长**：控制曲线平滑度（值越小曲线越平滑，但绘图越慢）

**示例：** 要看 `sin(x)` 在 0 到 4π 之间的图形：
- X 最小值设为 `0`
- X 最大值设为 `12.57`（≈ 4π）

### 4.5 使用预设函数

程序内置了 21 个常用函数，包括：

| 类别 | 预设示例 |
|------|---------|
| 基础函数 | `sin(x)`, `cos(x)`, `tan(x)`, `x^2`, `1/x` |
| 指数/对数 | `exp(x)`, `ln(x)` |
| 3D 函数 | `sin(x)*cos(y)`, `x^2+y^2` |
| FFT 预设 | `sin(2*pi*x)+0.5*sin(6*pi*x)` |

**使用方法：** 点击预设下拉菜单，选择一个函数即可。

### 4.6 参数系统

如果你的表达式中包含额外的字母变量（如 `a`, `b`, `k`），程序会自动检测并弹出参数输入框。

**示例：** 输入 `a*sin(b*x) + c`

程序会弹出三个输入框：
- `a` 的值
- `b` 的值
- `c` 的值

你可以实时修改参数值，曲线会动态更新！

## 6. 参数曲线绘图

参数曲线用参数 `t` 来定义 x 和 y 的坐标。

**操作步骤：**

1. 输入 x(t) 和 y(t) 的表达式
2. 点击 **"参数曲线绘图"** 按钮

**内置 10 种预设：**

| 名称 | x(t) | y(t) | 说明 |
|------|------|------|------|
| 圆 | `cos(t)` | `sin(t)` | 标准单位圆 |
| 椭圆 | `2*cos(t)` | `sin(t)` | x 方向拉伸 |
| 李萨如图形 | `sin(3*t)` | `cos(5*t)` | 经典物理图形 |
| 螺旋线 | `t*cos(t)` | `t*sin(t)` | 阿基米德螺线 |
| 心形线 | `16*sin(t)^3` | `13*cos(t)-5*cos(2t)-2*cos(3t)-cos(4t)` | 爱心形状 |
| 蝴蝶曲线 | `sin(t)*(e^cos(t)-2*cos(4t)-sin(t/12)^5)` | `cos(t)*(e^cos(t)-2*cos(4t)-sin(t/12)^5)` | 自然界曲线 |

**示例：手动画一个椭圆**

```
x(t) = 3*cos(t)
y(t) = 2*sin(t)
参数范围：0 到 2*pi
```

```

## 8. 隐函数绘图

隐函数曲线绘制形如 **f(x,y) = 0** 的方程（如圆、椭圆、双曲线、心形线等）。

**使用方法：**

1. 勾选 **"启用隐函数曲线"**
2. 输入包含 `x` 和 `y` 的表达式（如 `x^2+y^2-1`）
3. 调整分辨率（默认 200）
4. 点击 **"+ 添加曲线"** 或 **"绘图"**

**内置预设：**
- 圆：`x^2+y^2-1`
- 椭圆：`x^2/4+y^2-1`
- 双曲线：`x^2-y^2-1`
- 抛物线：`y-x^2`
- 双纽线：`(x^2+y^2)^2-(x^2-y^2)`
- 心形线：`(x^2+y^2-x)^2-(x^2+y^2)`
- 笛卡尔叶形线：`x^3+y^3-3*x*y`
- 三次曲线：`y^2-x^3+x`

**示例：绘制一个圆**

```
勾选"启用隐函数曲线"
输入：x^2+y^2-1
点击：绘图
```

**示例：绘制心形线**

```
勾选"启用隐函数曲线"
输入：(x^2+y^2-x)^2-(x^2+y^2)
点击：绘图
```

## 24. 数值计算

### 9.1 求值

**在指定点计算函数值：**

1. 输入表达式，如 `sin(x)`
2. 输入 x 的值，如 `1.57`（≈ π/2）
3. 点击 **"求值"**
4. 结果显示在结果区域：约 `1.0`

### 9.2 数值微分

**计算一阶导数 f'(x)：**

1. 输入表达式，如 `x^3`
2. 输入 x 的值，如 `2`
3. 点击 **"一阶导数"**
4. 结果：约 `12.0`（因为 3x² 在 x=2 处 = 12）

**计算二阶导数 f''(x)：**

1. 输入同样的表达式和 x 值
2. 点击 **"二阶导数"**
3. 结果：约 `12.0`（因为 6x 在 x=2 处 = 12）

**计算 n 阶导数：**

1. 输入表达式和 x 值
2. 输入阶数（如 5）
3. 点击 **"n阶导数"**

### 9.3 数值积分

**计算定积分 ∫f(x)dx：**

1. 输入被积函数，如 `x^2`
2. 输入下限 a = `0`
3. 输入上限 b = `1`
4. 点击 **"积分"**
5. 结果：约 `0.333`（= 1/3）

**示例：计算 ∫sin(x)dx 从 0 到 π**

```
表达式：sin(x)
下限：0
上限：3.14159
结果：约 2.0
```

### 9.4 方程求解

**求解 f(x) = 0 的根：**

1. 输入方程表达式，如 `x^2 - 4`（求解 x² - 4 = 0）
2. 输入初始猜测值或搜索范围
3. 点击 **"求根"**
4. 结果：`2.0`（或 `-2.0`，取决于猜测值）

**示例：求解 sin(x) = 0.5**

```
表达式：sin(x) - 0.5
初始猜测：1
结果：约 0.524（= π/6）
```

### 9.5 非线性方程组求解

**求解两个未知数的方程组：**

例如求解：
- f(x,y) = x² + y² - 1 = 0（单位圆）
- g(x,y) = x - y = 0（直线 y=x）

1. 输入第一个方程：`x^2 + y^2 - 1`
2. 输入第二个方程：`x - y`
3. 输入初始猜测 (x0, y0) = `(0.7, 0.7)`
4. 点击 **"求解方程组"**
5. 结果：x ≈ 0.7071, y ≈ 0.7071

### 9.6 极值查找

**求局部最小值：**

1. 输入表达式，如 `x^2 - 4*x + 3`
2. 输入搜索区间，如 `[-5, 5]`
3. 点击 **"求最小值"**
4. 结果：约 `2.0`（在 x=2 处取得最小值 -1）

**求局部最大值：**

1. 输入表达式，如 `sin(x)`
2. 输入搜索区间，如 `[0, 6]`
3. 点击 **"求最大值"**
4. 结果：约 `1.571`（≈ π/2，sin 的最大值点）

### 9.7 极限计算

**求双侧极限：**

1. 输入表达式，如 `sin(x)/x`
2. 输入趋近点，如 `0`
3. 点击 **"求极限"**
4. 结果：约 `1.0`

**求左极限（从左侧趋近）：**

```
表达式：1/x
趋近点：0
点击：左极限

## 24. ODE 求解器 (RK4)

**求解一阶常微分方程初值问题：**

方程形式：dy/dx = f(x,y)，已知 y(x₀) = y₀

**操作步骤：**

1. 输入微分方程右边的表达式，如 `-y`（求解 dy/dx = -y）
2. 输入初始条件：x₀ = `0`, y₀ = `1`
3. 输入求解区间终点：x_end = `5`
4. 输入步数（精度）：n_steps = `200`
5. 点击 **"ODE 求解"**
6. 显示解曲线图

**示例：求解 dy/dx = -y, y(0) = 1**

```
表达式：-y
x₀ = 0, y₀ = 1
x_end = 5, n_steps = 200
结果：y = e^(-x) 的数值解
```

## 24. 统计分布计算器

**计算概率分布的各种值：**

支持 6 种分布：

| 分布 | 参数 | 用途 |
|------|------|------|
| 正态分布 | μ (均值), σ (标准差) | 自然界中最常见的分布 |
| 学生 t 分布 | ν (自由度) | 小样本统计推断 |
| 卡方分布 | k (自由度) | 假设检验 |
| F 分布 | d1, d2 (自由度) | 方差分析 |
| 二项分布 | n (次数), p (概率) | 成功/失败实验 |
| 泊松分布 | λ (平均率) | 稀有事件计数 |

**操作步骤：**

1. 选择分布类型
2. 输入分布参数
3. 选择计算类型：
   - **PDF/PMF**：概率密度/概率质量函数
   - **CDF**：累积分布函数
   - **PPF**：逆累积分布函数（给定概率求分位数）
4. 输入变量值（或概率值）
5. 点击 **"计算"**

**示例：计算标准正态分布的 95% 置信区间**

```
分布：正态分布
μ = 0, σ = 1
计算 PPF(0.975)

```

## 24. 常见问题 (FAQ)

### Q1: 程序启动后提示找不到 calc_core.dll / .so

**A：** C 核心库未编译。请按以下步骤操作：

1. 确保已安装 GCC 编译器
2. 在项目根目录运行编译命令（见上方"从源码编译"）
3. 重新运行 Python 程序

### Q2: 绘图时曲线不显示

**A：** 可能原因：
- 表达式语法错误（如忘记写 `*` 号）
- Y 范围设置太小，曲线在视图外
- 尝试调整 X/Y 范围

### Q3: 方程求解失败

**A：** 可能原因：
- 初始猜测值离根太远
- 函数在该区间没有根
- 尝试换一个初始值或扩大搜索区间

### Q4: 3D 绘图卡顿

**A：** 尝试：
- 增大步长（如从 0.01 改为 0.05）
- 减小 X/Y 范围
- 关闭网格显示

### Q5: Android 安装 APK 失败

**A：** 可能原因：
- 系统版本低于 Android 7.0
- 未允许"未知来源"安装
- APK 文件下载不完整
